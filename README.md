# havensight-site

Public website for the **Havensight** Android game — a simple landing page and the
privacy policy required by Google Play and Google AdMob.

Static HTML/CSS, served via GitHub Pages.

- `index.html` — landing page
- `privacy.html` — privacy policy (the URL referenced by the Play listing and AdMob UMP)
- `style.css` — shared styling

## Live URL

Served via GitHub Pages (Deploy from branch → `main` / root), through the
custom domain `markhunsinger.com`:

- Site: https://markhunsinger.com/havensight-site/
- Privacy: https://markhunsinger.com/havensight-site/privacy.html

(Also reachable at https://markjhunsinger.github.io/havensight-site/)
